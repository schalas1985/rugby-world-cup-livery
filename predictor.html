predictor

This is the first wireframe for the predictor game. The idea is that a user can pick winners, assign scores, choose bonus points, and then watch the pools and knockout bracket update automatically.

For now this page is about structure only: interactive inputs, pool cards, and a knockout board that we can gradually connect to the real tournament rules.
