# Rugby World Cup Livery

A static website tribute to Rugby World Cup history, beginning with a landing page and a champions page.

## Current pages

- `index.html` - landing page
- `champions.html` - champions tribute page
- `styles.css` - shared styling

## Why this version is simple

This project is intentionally built with plain HTML and CSS so the focus stays on learning Codex and shipping quickly, without needing paid tools or a JavaScript build setup.

## Publish plan

1. Create a GitHub repository.
2. Add these files.
3. Push to GitHub.
4. Enable GitHub Pages from the repository settings.
5. Use the `main` branch and the `/ (root)` folder as the source.
